🏗️ System Architecture

Application Layer

Quarkus Application: REST API with intentional memory leak for demonstration. Includes health checks, metrics endpoints, and Instana Java agent integration.

Monitoring Layer

Instana Agent (DaemonSet): Monitors JVM metrics, memory usage, GC activity. Sends data to Instana SaaS for analysis and alerting.

Intelligence Layer

Bob AI Agent: Node.js/TypeScript microservice with alert handler, code analyzer, fix generator, and GitHub integration. Uses MCP for tool connectivity.

CI/CD Layer

OpenShift Pipeline (Tekton): Automated build, test, and deployment pipeline triggered by GitHub webhooks.

Deployment Layer

GitOps (ArgoCD): Declarative continuous deployment with automatic synchronization and self-healing capabilities.

⏱️ Detailed Process Timeline

1

T+0s: Memory Leak Triggered

Application endpoint called, static collection starts accumulating data without cleanup mechanism.

private static final Map> LEAKED_MEMORY = new ConcurrentHashMap<>(); // Static modifier prevents garbage collection
2

T+5min: Instana Detection

Instana agent detects memory usage exceeding 80% threshold for sustained period.

3

T+5min 10s: Alert Webhook Sent

Instana SaaS sends webhook to Bob AI Agent with alert details and metrics.

4

T+5min 15s: Bob Analysis Starts

Bob validates alert, fetches additional context from Instana, identifies affected application.

5

T+5min 30s: Code Retrieved

Bob fetches source code from GitHub repository using GitHub API.

6

T+5min 45s: Pattern Analysis

Bob's code analyzer scans for memory leak patterns, identifies static collection issue.

7

T+6min: Fix Generated

Bob generates fix by removing static modifier, validates syntax, creates explanation.

// BEFORE: static collection (memory leak) private static final Map> LEAKED_MEMORY = new ConcurrentHashMap<>(); // AFTER: instance variable (allows GC) private final Map> leakedMemory = new ConcurrentHashMap<>();
8

T+6min 30s: Pull Request Created

Bob creates branch, commits fix, opens PR with detailed description and testing checklist.

9

T+7min: Pipeline Triggered

GitHub webhook triggers OpenShift Pipeline for automated build and test.

10

T+15min: Build Complete

Pipeline completes: code fetched, Maven build successful, tests passed, image pushed to registry.

11

T+16min: GitOps Sync

ArgoCD detects new image tag, syncs configuration, initiates rolling update.

12

T+20min: Deployment Complete

Fixed application deployed, health checks passing, memory metrics returning to normal.

13

T+30min: Verification

Instana confirms memory leak resolved, alert cleared, system stable.
